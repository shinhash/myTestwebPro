의사 클래스(pseudo-class, 가상클래스):  
클래스가 정의된 것처럼 간주. 
상태나 구조에 의해서 선택이 이루어진다 

상태
	a:link		처음상태
	a:visited	한번 방문한 상태
	a:hover		마우스가 올려져 있는 상태
	a:active	마우스를 down 하는 상태




	a:link { color: blue; } 처음
	a:visited { color: green; } 방문한 적이 있다
	a:hover { color: red; } 마우스 올려 놨을때
	a:active {color : pink; } 마우스 down 상태




	td:nth-child(2n)	-> 0 2 4 6 8 
	td:nth-child(2n+1)	-> 1 3 5 7 9
	td:nth-child(2n+2)	-> 2 4 6 8
