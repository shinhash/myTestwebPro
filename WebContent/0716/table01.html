작성팁
			
			=================================================
			
			table		: 표형태의 데이터를 표시
			tr(row)		: 행, 줄
			td(data)	: 열, 칸
			th(header)	: td대신 제목 표시
			
			스타일
			table		: border, border-collapse
			tr			: width, height, text-align
			td			: background
			th			: width, height
			
			=================================================
			
			table 자식(>)
			1. table > tr		2. table > td
			3. tr > th			4. tr > td
			
			table 후손(공백)-자식을 포함
			1. table tr		2. table td
			3. tr th 		4. tr td
